Resultado A − B

Procedimiento paso a paso

Se usa la resta binaria con préstamo, igual que en decimal pero en base 2.

Resultado negativo: El minuendo A es menor que el sustraendo B, por lo que A − B es negativo. Se calcula  |B − A|  y se antepone el signo −.

¿Cómo funciona la lógica?

Reglas básicas

0 − 0 = 0
1 − 0 = 1
1 − 1 = 0
0 − 1 → préstamo
10₂ − 1 = 1

Algoritmo con préstamo

  1. Alinea los bits de derecha a izquierda (del menos al más significativo).
  2. Resta columna a columna. Si el bit superior es menor, pide préstamo a la columna de la izquierda.
  3. Pedir préstamo equivale a sumar 2 al bit actual y restar 1 al bit de la izquierda.
  4. El resultado se lee de izquierda a derecha.

Casos negativos

Esta calculadora compara los valores decimales de A y B. Si A < B, el resultado es negativo. Se muestra el signo − y el procedimiento usa |B − A| para ilustrar la mecánica del préstamo de forma clara. No se usa complemento a 2 para mostrar el procedimiento, pero sí se indica cuándo el resultado es negativo.
